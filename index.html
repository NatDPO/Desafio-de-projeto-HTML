O que é?

Introdução

Antes de mais nada, precisamos entender o significado da palavra HTML "HyperText Markup Language", por mais que o nome por extenso possua o termo 'linguagem', HTML não é uma linguagem de programação, é uma ferramenta de marcação que serve para estruturar seu site/página.

Ele organiza o conteúdo com tags que definem títulos, parágrafos, imagens e links. Basicamente, constrói a base de um site, mas ganha mais funcionalidade com CSS e JavaScript, criando experiências completas de interatividade com clientes e usuários.

A estrutura básica de um HTML (usualmente utilizada por todo o mundo), seria HTML(título do seu arquivo não é necessariamente HTML, mas sim o que você escolher) - HEAD - BODY - HTML.

Existem muitas plataformas que é disponibilizada para executar projetos de HTML, sendo pagas ou open source, uma batstante utilizada que seria open source é o Visual Studio Code.

Como é realizado essa comunicação de código e navegação?

Quando acessamos um site, muitas vezes não paramos para refletir sobre como acontece a interatividade entre o usuário e a tecnologia. No entanto, essa troca é facilitada pela combinação entre o front-end e o back-end, ou seja, entre o cliente e o servidor.

O front-end lida com a parte visível e interativa da aplicação, ou seja, é a ponte inicial entre o usuário e a máquina. Pode-se pensar nele como responsável pelo "efeito estético" do site ou aplicativo, proporcionando uma experiência visual e interativa. Já o back-end, por sua vez, é responsável pela parte "invisível", que trata de processar e enviar as informações solicitadas. Ele executa tarefas como armazenar dados ou realizar ações no servidor. Um exemplo simples disso seria a ação de reagir a uma publicação nas redes sociais. Quando você clica no "coraçãozinho" para curtir, essa informação é enviada ao servidor, que registra sua interação, associando você àquele conteúdo. Esse tipo de operação ocorre constantemente nos bastidores dos sites e aplicativos, tornando a experiência de navegação fluída e intuitiva.

Caso o sistema não compreenda corretamente o que o usuário deseja fazer, uma mensagem de erro é gerada, o que acontece graças ao código que o back-end já tem preparado para lidar com esses casos. Em resumo, toda interação que temos com sites e aplicativos envolve uma complexa colaboração entre o front-end, o back-end, e o servidor, garantindo que o processo de comunicação entre usuário e sistema aconteça de forma eficiente.

O que são TAG's?

No HTML, as tags são elementos essenciais para criar a estrutura de uma página web. Elas são usadas para organizar e definir diferentes tipos de conteúdo, como textos, imagens, links e muito mais. As tags são representadas por sinais de menor e maior (< e >), geralmente formando pares: uma tag de abertura (por exemplo, < p >) e uma tag de fechamento (como < /p>), que delimitam o início e o fim de um elemento.

Algumas tags, como a < img> e a < br>, são auto-fechantes, ou seja, basicamente não precisam de uma tag de fechamento, pois executam uma ação simples, como inserir uma imagem ou criar uma quebra de linha. Além disso, as tags podem incluir atributos, que são propriedades adicionais que alteram o comportamento ou a aparência do elemento. Os atributos são especificados dentro da tag de abertura, no formato atributo="valor". Por exemplo, o atributo src na tag < img> define o caminho da imagem, enquanto o atributo href na tag indica o destino de um link.

Essas tags e seus atributos permitem criar páginas web dinâmicas e interativas, organizando o conteúdo de maneira clara e eficiente para os navegadores. Em resumo, as tags no HTML são fundamentais para estruturar o conteúdo e permitir que ele seja exibido corretamente na web.

O que são atributos?

Os atributos são um complemento das Tag's, são características extras que fornecem informações adicionais sobre algum elemento. Eles são usados para controlar/ajustar, a aparência ou a funcionalidade de um item na página. Os atributos são inseridos dentro da tag de abertura e seguem o formato atributo="valor", permitindo modificar/personalizar o comportamento do elemento.

Cada atributo tem uma função específica. Por exemplo, na tag , o atributo src define o caminho para o arquivo da imagem, enquanto o atributo alt fornece uma descrição alternativa da imagem. Da mesma forma, a tag , responsável por links, usa o atributo href para especificar o destino do link.

Além de influenciar o comportamento do conteúdo, os atributos podem ser usados para identificar elementos únicos, aplicar estilos ou até mesmo manipular a página com JavaScript. Atributos como class e id são comuns para aplicar estilos com CSS ou para referenciar elementos em scripts.

  1. href: Define o endereço de destino de um link.
  2. src: Define o caminho para o arquivo de uma imagem.
  3. class: Aplica uma classe CSS para estilizar um elemento.
  4. id: Atribui uma identificação única a um elemento (ex: ).

    5. Esses atributos tornam possível personalizar e controlar melhor os elementos em uma página, facilitando a criação de interfaces mais interativas e funcionais. Existem muitos outros atributos e TAG's, que nos ajudam a organizar e monstar nossa estrutura de WEB, que podem ser verificados em sites que possuem essa listagem.

Como montar meu site - básico

Agora que entendemos um pouco como um site funciona, vamos recapitular algumas informações.

Já entendemos que o HTML realiza a estrutura do seu site, entretanto conforme o avanço das tecnologias mundiais, foi surgindo novas tecnologias que abrangem e interagem entre elas mesma, o CSS por exemplo é comumente utilizado para estilizar todo o conteúdo e também o JavaScript.

Ele (CSS) define como os elementos do HTML aparecerão na tela, controlando espaçamentos, cores, fontes, tamanhos entre muitas outras coisas, ou seja, basicamente cuida do layout da tela, tornando a tela mais atrativa e agradável para o usuário.

Além disso é possível criar um design responsivo, ou seja, garantir que o site seja exibido em qualquer dispositivo sem lacunas indesejadas.

Já com o JavaScript por ser uma linguagem de programação, o mesmo auxilia na interatividade do usuário com o site.

Com o JavaScript, é possível realizar ações dinâmicas em uma página, como atualizar seu conteúdo sem precisar recarregar a página, validar formulários, criar animações e até interagir com o servidor de maneira assíncrona.

Em outras palavras, o JavaScript torna um site estático em uma experiência interativa e responsiva, proporcionando uma navegação mais fluída e envolvente.

Para montar um site, precisamos ter algumas ferramentas básicas e posteriormente vamos implementando novas tecnologias para nos auxiliar facilitando a fluidez de nosso site.

Caso utilizemos o visual studio code, precisamos primeiramente criar uma pasta em nosso computador onde "ficará nosso site", após isso começamos a monstar a estrutura base do nosso site, sabemos que tudo que estiver entre HTML é um HTML, tudo que estiver em HEAD seão os títulos que ficam na ABA do navegador e tudo que estiver em BODY é o que irá aparecer para o usuário.

No BODY é onde colocaremos nossos textos, tabelas, imagens e afins, utilizando as tags e atributos. Posteriormente com nossos site montado podemos ajustar e personalizar o mesmo pelo CSS ou pelo próprio JavaScript.
